Estratégias que fazem a diferença

Ninguém fica de fora

Pequenos detalhes que tornam o aprendizado mais humano e eficaz para quem está começando do zero.

🏆

Líder da Turma

Quem terminar o desafio primeiro é nomeado Ajudante Oficial do dia. Empodera quem aprendeu rápido e dá fôlego para o professor.

⏱

Cronômetro na Tela

Durante os desafios, um timer visível cria clima de gincana. A turma foca mais e se diverte mais.

🎙

Compartilhamento de Tela

Um aluno diferente a cada dia mostra como resolveu. Valoriza o esforço e cria aprendizado coletivo.

💬

Lidando com a Frustração

"Isso faz parte! Até os profissionais esquecem de fechar uma tag. Vamos achar o erro juntos?" — o erro normalizado vira aprendizado.

🤖

IA como Copiloto

A partir da Aula 7, a IA entra como ferramenta — não como muleta. Gerar código, entender código, corrigir código.

📱

Inclusão Digital

Quem tem só celular acompanha no CodePen. Quem não instalou o VS Code continua. A inclusão é parte do curso.

6
CSS
A Caixa Mágica — Box Model e Flexbox
29/04 — Quarta · 14h às 17h
▼

Compreender o Box Model e usar Flexbox para posicionar elementos lado a lado — os dois conceitos que desbloqueiam o layout moderno.

14:00–14:35
Retrospecto + Aquecimento — O Código Quebrado
Voluntário explica Google Fonts e text-align. Desafio: encontrar erros de sintaxe num CSS proposital.
14:35–15:20
Teoria ativa — O quadro na parede (Box Model)
Content = foto. Padding = moldura branca. Border = madeira do quadro. Margin = distância entre quadros. Ver no DevTools.
15:20–15:45
Prática guiada — O botão perfeito
Transformar tag <a> em botão com padding, background-color e border-radius. Antes e depois na tela.
15:45–16:00
Intervalo
—
16:00–16:45
Laboratório — Galeria de Fotos com Flexbox
display:flex. Colocar 3 imagens lado a lado e centralizadas. Adicionar o botão do lab anterior abaixo das imagens.
16:45–17:00
Fechamento
"Se entenderam Box Model + Flexbox, já entendem 60% dos sites modernos." Pedir ideias para o projeto final.
⚡ Desafio — O Código Quebrado

Encontre os 2 erros: h1 { color: blue font-size: 20px; } — Primeiro que achar e corrigir os dois avisa no chat!

🎣 Gancho para a próxima aula
